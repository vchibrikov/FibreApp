# FibreApp
FibreApp is a repository of Pyhton code, Tensorflow and CoreML models for machine learning image classification. Repository created for the project grant nr NdS-II/SP/0258/2023/01, supported by the Minister of Science and Higher Education of Poland as a part of “Nauka dla Społeczeństwa II” program. More information on the project: www.fibreapp.pl
